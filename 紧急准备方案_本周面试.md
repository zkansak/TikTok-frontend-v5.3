#### 1. JavaScript 核心（2小时复习）

**闭包（必考）**
```javascript
// 防抖函数（你的项目用了）
function debounce(func, delay) {
  let timer;
  return function(...args) {
    clearTimeout(timer);
    timer = setTimeout(() => func.apply(this, args), delay);
  };
}

// 面试官可能问：为什么用闭包？
// 答：保存 timer 变量，避免全局污染，实现函数复用
```

**this 指向（必考）**
- 普通函数：指向调用者
- 箭头函数：指向定义时的上下文
- call/apply/bind 的区别

**异步编程（必考）**
```javascript
// Promise 链式调用
fetch('/api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// async/await
async function fetchData() {
  try {
    const res = await fetch('/api/data');
    const data = await res.json();
    return data;
  } catch (err) {
    console.error(err);
  }
}
```

#### 2. React 核心（3小时复习）

**useState**
```javascript
const [count, setCount] = useState(0);
// 函数式更新
setCount(prev => prev + 1);
```

**useEffect（必考）**
```javascript
// 空数组：只在挂载时执行一次
useEffect(() => {
  // 初始化逻辑
}, []);

// 有依赖：依赖变化时执行
useEffect(() => {
  // 副作用逻辑
}, [count]);

// 清理函数
useEffect(() => {
  const timer = setInterval(() => {}, 1000);
  return () => clearInterval(timer); // 清理
}, []);
```

**useCallback 和 useMemo（必考）**
```javascript
// useCallback：缓存函数
const handleClick = useCallback(() => {
  // ...
}, [deps]);

// useMemo：缓存值
const expensiveValue = useMemo(() => {
  return computeExpensiveValue(a, b);
}, [a, b]);
```

**useReducer（你的项目用了）**
```javascript
// 适合复杂状态管理
const [state, dispatch] = useReducer(reducer, initialState);

function reducer(state, action) {
  switch (action.type) {
    case 'INCREMENT':
      return { ...state, count: state.count + 1 };
    default:
      return state;
  }
}
```

#### 3. 浏览器 API（你的项目重点）

**IntersectionObserver（你的项目核心）**
```javascript
// 你的项目用了这个API实现无限滚动
const observer = new IntersectionObserver(
  (entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        // 触发加载更多
      }
    });
  },
  {
    threshold: 0.1,
    rootMargin: '100px'
```

- Q: 为什么用 IntersectionObserver 而不是监听 scroll 事件？

### 📝 技术面常见问题（快速准备）

#### Q1: 闭包的应用场景？
- 防抖节流
- 模块化（私有变量）
- 函数工厂

#### Q2: useEffect 的依赖数组？
- `[]`：只在挂载时执行一次
- `[count]`：count变化时执行
- 无依赖：每次渲染都执行（不推荐）

#### Q3: useCallback 和 useMemo 的区别？
- useCallback：缓存函数，避免子组件不必要的渲染
- useMemo：缓存值，避免重复计算

#### Q4: 如何优化长列表性能？
- 虚拟窗口：只渲染可见区域
- React.memo：组件缓存
- useMemo：值缓存
- 懒加载：按需加载数据

#### Q5: 如何实现防抖？
```javascript
function debounce(func, delay) {
  let timer;
  return function(...args) {
    clearTimeout(timer);
    timer = setTimeout(() => func.apply(this, args), delay);
  };
}
```
